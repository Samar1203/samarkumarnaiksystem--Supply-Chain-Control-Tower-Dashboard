# samarkumarnaiksystem--Supply-Chain-Control-Tower-Dashboard
A UI/UX design project for a Supply Chain Control Tower Dashboard.


## Overview

Supply Chain Control Tower Dashboard is a UI/UX design project created in Figma for logistics and operations management.

The platform helps organizations monitor shipments, warehouses, inventory levels, fleet performance, and operational KPIs through a centralized dashboard.
